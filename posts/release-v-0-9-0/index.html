Forestry Bar

Release Version 0.9.0

April 5, 2023

作为一个大学生创业的实体项目，我们在很短的时间里实现了盈利，同时扩展了我们的业务范围，为大家带来了欢聚一堂的快乐时光

现在，我们自豪且兴奋地宣布：我们即将进入每晚都开张的营业模式

除此之外，林吧酒馆的顾客都应该遵守我们的准则，总结起来无非以下几点:

  1. 所有顾客都应该遵守《北京市控制吸烟条例》，我们要求您不得在林吧酒馆内吸烟，包括传统香烟、水烟和电子烟等；如果您是吸烟者，我们要求您吸烟时前往室外

  2. 所有的顾客必须遵守中国的法律和北京市的地方法规，无论您是否是中华人民共和国的公民

  3. 尊重他人，您应该避免冒犯他人，尊重林吧内其他顾客的习惯，爱好，信仰，性别认知，性取向等

  4. 禁止劝酒，这包括言语上和动作上，无论您是有意或无意，这都是不被允许的；林吧希望每位顾客都有自己选择饮酒量的权利，并且愿意保护这项权利不被侵犯

  5. 行为举止符合公序良俗

林吧致力于为大家提供舒适的闲谈放松环境，如果您在林吧因为任何情形感到被冒犯，请您无需顾虑立即联系店员寻求帮助；如果您在林吧受到店员冒犯，请立即要求联系林吧酒馆的主要负责人，或者透过我们的微信群组进行反馈

我们将不留余力地严肃处理这类事件

这份准则将作为正式准则发布前的临时准则使用

特此鸣谢： 人文学院法学研究会会长 Pan さん

他指出临时准则中部分用语不符合规范并给出解决方案
